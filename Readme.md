## What ?

## How to Run it ?

First you need node and git installed after that run:

_npm_

```sh
git clone https://github.com/SaraVieira/awesome-talks
cd awesome-talks
npm i
npm run dev
```

_yarn_

```sh
git clone https://github.com/SaraVieira/awesome-talks
cd awesome-talks
yarn
yarn dev
```
